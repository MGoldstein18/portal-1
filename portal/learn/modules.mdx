## Intro

With thirdweb you can mint NFT's, create drops and build your own marketplace. All of this is done inside Projects and Modules. In this guide we'll deploy a module on your thirdweb project. If you donâ€™t know what **Projects** and **Modules** are, then [click here.](/portal/learn/introduction)

## Dashboard setup

### Create a Project

![My_project_2.png](/assets/portal/learn/how-to-create-and-deploy-projects-modules/My_project_2.png)

Pick any module you want. In this tutorial we want to create a drop, so we need the 'Drop' module for this.

![Drop.png](/assets/portal/learn/how-to-create-and-deploy-projects-modules/Drop.png)

Go ahead and fill in a name for this module and optionally you can add a 'Royalty'.

![Screenshot 2021-12-04 at 14.58.38.png](/assets/portal/learn/how-to-create-and-deploy-projects-modules/add-module.png)

You will be asked to confirm two transactions again to create the module.

![Untitled](/assets/portal/learn/how-to-create-and-deploy-projects-modules/adding-module.png)

![Untitled](/assets/portal/learn/how-to-create-and-deploy-projects-modules/metamask4.png)

![Untitled](/assets/portal/learn/how-to-create-and-deploy-projects-modules/metamask5.png)

</ImageSideToSide>

The module is created and deployed on to the blockchain.
