## What you get

This guide gives you a simple React **component** to get the balance and a button to mint some tokens. Also [here](https://github.com/thirdweb-dev/examples) is a link to our Github page with examples.

## Dependencies

Quick heads up, in these examples, the thirdweb sdk is integrated with React **components**, but we're â€˜just' using javascript.

Make sure to have the dependencies installed 

```jsx:App.js
npm install @thirdweb-dev/sdk ethers

npm install @thirdweb-dev/react @thirdweb-dev/hooks
```

## The App.js - Connect your wallet

In your `App.jsx` we import our **component** for the `connect wallet` button and `create token` button.

```jsx
import { ThirdwebProvider } from '@3rdweb/react';

import './App.css';
import TokenComponent from '../components/tokencomponent';
import Connect from '../components/connect'

const supportedChainIds = [1, 4, 137, 250, 43114, 80001];

function ExampleApp() {

  return (
    <ThirdwebProvider desiredChainId={supportedChainIds}>
      <Connect />
      <TokenComponent />
    </ThirdwebProvider>
  );
}
export default ExampleApp;
```

## Add The thirdweb Connect Component

Create a new folder called components that will host all custom components of your app. After that, create a new file called connect.js. The component is very simple and only consists of the following code:

```js:connect.js
import { useDisconnect, useMetamask } from "@thirdweb-dev/react";

export const Connect = () => {
  const connectWithMetamask = useMetamask();
  return <button onClick={connectWithMetamask}>Connect Metamask Wallet</button>;
};
```

## The tokencomponent.js - The mint token button
Create a file called `tokencomponent.js` inside your components folder.
In your `tokencomponent.js` file we will write the code for the button and get balance. If you want to know more about the token contract you can find a guide [here](/guides/create-your-currency-with-typescript-sdk).

The cool thing about this component, is that it's using the `connect` component to authorise the transaction ðŸ˜„.

```jsx:tokencomponent.js
import { useWeb3 } from "@thirdweb-dev/hooks";
import { useEffect, useMemo, useState, useCallback } from "react";
import { ThirdwebSDK } from "@thirdweb-dev/sdk";
import { ethers } from "ethers";

//use the wallet instead of private keys
const TokenComponent = () => {
  const { provider } = useWeb3();
  const [balance, setBalance] = useState(0);

  const sdk = useMemo(() => {
    if (provider) {
      return new ThirdwebSDK(provider.getSigner())
    }

    return undefined;
  }, [provider]);

//instantiate the sdk
  const tokencontract = useMemo(() => {
    if (sdk) {
      return sdk.getToken("0x0d5fb8942eEa62093944F3e91C6Ac4e584336741")
    }

    return undefined;
  }, [sdk]);

//get the balance from our token contract
  useEffect(() => {
    const getBalance = async () => {
      const tokenBalance = await tokencontract.balance()
      setBalance(tokenBalance.displayValue); 
    }

    if (tokencontract) {
      getBalance()
    };
  }, [tokencontract]);

//mint our token. remember, ethereum deals different with numbers.
// that's why we need to convert our amount
  const amount = ethers.utils.parseUnits("1000", 18);
  const onClick = useCallback(() => {
    

    tokencontract.mint(amount);

  }, [amount, tokencontract]);

  return (
    <div>
      Balance: {balance}
      <br />
        <button class="big-button" onClick={onClick}>Get me a 1000 Coins!</button>

    </div>
  );
};
export default TokenComponent;
```
